# RoleChat 功能图标清单

> 共 **15 个**功能图标，覆盖底部导航、操作按钮、消息交互、角色创建四大场景。
> 设计规范依据 `项目开发文档.md` §4.

---

## 设计规范速查

| 项目 | 值 |
|------|-----|
| 风格 | 极简线性（描边为主，必要时部分填充） |
| 基础画布 | 24×24 px（@1x），导出 48×48 px（@2x） |
| 线条粗细 | 4 pt（48px 画布下 ≈ 2px） |
| 外圆角 | 2 px |
| 内圆角 | 1 px |
| 默认态 | 描边 `#8E8E93`，无填充 |
| 活跃态 | 填充 `#1A73E8` |
| 禁用态 | 描边 `#D1D1D6`，整体透明度 40% |
| 导出格式 | SVG（矢量源文件）+ PNG @1x/2x/3x |

## 汇总

| 分类 | 数量 | 图标 |
|------|------|------|
| 底部导航 | 4 | 对话、角色、发现、我的 |
| 列表操作 | 4 | 新建对话、搜索、删除、收藏 |
| 对话界面 | 2+1 | 发送、角色切换、（返回） |
| 消息操作 | 4 | 重新生成、复制、点赞、踩 |
| 角色创建 | 1+3 | 世界观、（角色设定、用户身份、保存） |
| **核心图标** | **15** | 满足教师最低要求 |

额外辅助图标（括号内）：返回、角色设定标签、用户身份标签、保存 —— 共 4 个。全部完成则总计 **19 个 SVG 资源**。
