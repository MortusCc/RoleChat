# 实验二 产品图标设计与制作 —— 实验报告

> **项目名称**：RoleChat —— 通用 AI 角色扮演轻问答网页  
> **设计工具**：草图手绘 + SVG 代码实现  
> **设计日期**：2026 年 5 月

---

## 一、设计需求

RoleChat 是一款 AI 角色扮演对话平台。需要一枚品牌产品图标（App Icon），传达「AI 对话」+「角色扮演」双重产品基因。图标需在 16×16 到 1024×1024 像素范围内保持可识别性。

---

## 二、设计定位

### 2.1 品牌分析报告

#### 2.1.1 关键词法提取

| 层级 | 关键词 | 提取逻辑 |
|------|--------|----------|
| 核心功能 | AI 对话、角色扮演 | 产品核心动作 |
| 情感感知 | 沉浸、陪伴、友好 | 用户情绪体验 |
| 视觉联想 | 聊天气泡、字母 R、面具 | 可对标的视觉符号 |

收敛为 3 个核心关键词：**对话 · 角色 · AI**

#### 2.1.2 竞品对比

| 竞品 | 图标类型 | 启示 |
|------|----------|------|
| ChatGPT | 中心对称星形 | 避免纯科技冷感 |
| Character.AI | 彩色人物面孔 | 角色感值得借鉴 |
| 微信 | 绿色对话气泡 | 气泡是聊天最直观的隐喻 |
| Slack | 四色井字 | 品牌色一致性重要 |

**差异化定位**：不做纯工具图标（ChatGPT 冷感星形），不做纯社交（微信单气泡），而是用**品牌首字母 R 承载对话气泡隐喻**——字母即功能。

#### 2.1.3 用户画像

| 属性 | 描述 |
|------|------|
| 年龄 | 18–28 岁 |
| 兴趣 | 二次元、游戏、同人创作、AI 工具 |
| 场景 | 课余/通勤时打开，进行沉浸式角色扮演对话 |
| 审美偏好 | 现代简洁、有品牌辨识度、不要过于卡通 |

#### 2.1.4 情绪板

收集关键词：**蓝色渐变**（科技 + 信任）、**粗细线条对比**（层次与节奏）、**白色单色图形**（iOS 风格）、**微投影**（轻拟物质感）。参考品牌：Stripe、Notion。

### 2.2 设计方向探索

| 方向 | 核心图形 | 结论 |
|------|----------|------|
| 方向 1：RC 字形 | 品牌首字母 R，粗线条 + 对话圆点 | ✅ 唯一性最强，字母即品牌 |
| 方向 2：气泡 + 面具 | 白色气泡内嵌半透明面孔 | ❌ 小尺寸面具不可辨认 |
| 方向 3：AI 星火 | 对称星形 + 气泡轮廓 | ❌ 与 ChatGPT 过于雷同 |

**选定方向 1** —— R 字形设计。品牌首字母具有不可替代的唯一性，且可以通过造型手法同时承载「对话气泡」隐喻。

## 三、草图设计

### 3.1 设计思路

R 由三个笔划组成，各自承担不同的语义角色：

| 笔划 | 形态 | 隐喻 |
|------|------|------|
| 竖线 | 直角梯形（上直下斜），描边 50px | 对话气泡的尾巴——上宽下窄、底部倾斜 |
| 弧线头 | 几何折线（无贝塞尔曲线），描边 100px | R 的头部，直线段构成硬朗的现代字形 |
| 斜腿 | 直线，描边 100px | R 的字形识别核心，保持字母辨识度 |

三个横向排列的白色圆点置于弧线内部负空间，模拟聊天中"对方正在输入…"的三点跳动动效。

### 3.2 笔划粗细的层次设计

- **弧线头 (100px)** 和 **斜腿 (100px)** ：粗线条确保小尺寸下 R 字形仍然清晰可辨
- **竖线/梯形 (50px)** ：减半的描边粗细，视觉上"后退"到第二层级，不争夺弧线和斜腿的主体地位
- **圆点 (r=20)** ：更小的元素，透明度由 0.9→0.55→0.25 递减，静态画面上营造动感

这种粗细对比（100 / 50 / 20）在单一白色图形中建立了三层视觉节奏。

### 3.3 梯形竖线

竖线没有用传统的等宽直线，而是设计为**直角梯形**：

- **上直角**：顶部水平，与弧线头的起笔处（x=-130）精确对齐
- **下斜边**：底部向右下倾斜，视觉上与气泡尾巴的指向性一致
- **左长右短**：左侧（外缘）更长，右侧（内缘）更短，形成由外向内收缩的趋势——这正是对话气泡尾巴的典型轮廓

R 的竖线不是"一根线"，而是一个**具有方向性的几何体**。它承担着"气泡尾巴"的隐喻功能，同时保持字母 R 的结构完整性。

### 3.4 几何折线弧线的设计选择

弧线头没有使用传统贝塞尔曲线（`C` 命令），而是用纯直线段（`L` 命令）构成几何多边形。这种选择带来两个效果：

- **现代感**：直线转折比曲面更硬朗，契合移动互联网产品的简洁调性
- **结构对齐**：折线起点 x=-130 与梯形右边缘精确重合，R 的连接处无缝隙

## 四、低保真设计

### 4.1 矢量骨架

| 元素 | 形状 | 坐标 / 参数 |
|------|------|-------------|
| 背景 | 圆角矩形 rx=225 | 1024×1024 |
| 竖线 | 直角梯形，stroke=50，fill=none | (-180,-330)→(-130,-330)→(-130,80)→(-180,350) |
| 弧线 | 几何折线（L 命令），stroke=100 | (-130,-320)→(180,-320)→(260,-200)→(260,-50)→(260,70)→(170,110)→(40,110)→(-130,110) |
| 斜腿 | 直线，stroke=100 | (40,110)→(280,320) |
| 圆点 | 3× circle r=20 | 圆心 (-20,-90) / (40,-90) / (100,-90) |

### 4.2 色彩

| 色值 | 用途 |
|------|------|
| `#1557B0` → `#2B7DE1` | 背景 45° 渐变 |
| `#FFFFFF` | 前景图形 |
| `0.9 / 0.55 / 0.25` | 圆点透明度递减 |

## 五、高保真设计（视觉稿）

### 5.1 最终效果

在低保真骨架上添加 `feDropShadow`（dy=6, blur=8, #0A3060@30%），白色 R 字形从蓝色背景上浮起，获得轻拟物的层次感。

### 5.2 设计亮点

- **粗细节奏**：100px / 50px / 20px 三层笔划粗细，在单一白色图形中建立视觉层次
- **梯形即尾巴**：直角梯形竖线不是装饰，而是对话气泡尾巴的结构化表达——左长右短、下斜边的形态天然带有"指向性"
- **几何折线**：弧线头用纯直线段代替贝塞尔曲线，赋予字形现代硬朗的气质
- **精确对齐**：弧线起点 x=-130 与梯形右边缘正好重合，R 的连接处无断缝
- **静态动效**：三圆点透明度递减，在静态图标中模拟"对方正在输入"的动态暗示

## 七、自评

| 维度 | 权重 | 说明 |
|------|------|------|
| 品牌契合度 | 30% | 字母 R = 品牌名称，梯形 = 气泡尾巴，圆点 = 对话状态——三重语义合于一个图形 |
| 平台规范性 | 25% | 1024px 画布、iOS 超椭圆圆角、三尺寸 PNG 导出 |
| 视觉表现力 | 20% | 粗细笔画对比 + 透明度渐变动效 + 投影层次 |
| 技术完整性 | 15% | 纯 SVG 实现，5 组图形元素，命名与导出规范 |
| 创新性 | 10% | 梯形竖线代替等宽直线 + 几何折线代替贝塞尔曲线，以笔画形态本身承载功能隐喻 |
